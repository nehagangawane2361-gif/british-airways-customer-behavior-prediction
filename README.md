# ✈️ British Airways Customer Behavior Prediction

## 📌 Project Overview
This project focuses on predicting customer booking behavior using machine learning techniques on the British Airways dataset.

## 🎯 Problem Statement
The goal of this project is to understand the factors that influence whether a customer completes a booking or not.

## 🧠 Approach
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Logistic Regression Model
- Model Evaluation
